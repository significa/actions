# Shared GitHub actions workflows by Significa

Reusable GitHub actions workflows for common CI/CD tasks.

## Available actions

| Action | Description |
|--------|-------------|
| [Deploy static site to AWS](docs/aws-static-site.md) | Build and deploy static sites to AWS S3 + CloudFront |
| [Deploy container to fly](docs/container-on-fly.md) | Deploy containerized apps to Fly.io with staging/production environments |
| [Build and push docker image](docs/docker-image.md) | Build and push Docker images to container registries |
| [Test and publish elixir library](docs/elixir-library.md) | Test Elixir libraries and publish to Hex |
| [Test and publish NPM library](docs/npm-library.md) | Build, test, and publish NPM packages |
| [Test and publish Python package](docs/python-package.md) | Test and publish Python packages to PyPI |
| [Release Drafter](docs/release-drafter.md) | Create draft releases with auto-generated release notes |
| [Build and deploy a Vercel app](docs/vercel-app.md) | Build, test, and deploy applications to Vercel |
